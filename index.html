Формулы расчета количества шпал

Основная формула расчета шпал:
N = Math.ceil((L × E) / 1000)

где:
N - количество шпал (округляется в большую сторону)
L - протяженность участка (м)
E - эпюра шпал (шт/км)

Логика учёта криволинейных участков:
if (секция === "кривая" ИЛИ секция === "комбинированная") {
  if (радиус > 600) {
    записать_протяженность_в_столбец_кривых
  }
}

где:
• Для кривых участков: записывается вся протяженность
• Для комбинированных: записывается только длина кривой части
• Учитываются только участки с радиусом более 600 метров
